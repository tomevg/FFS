# Очередь работ по СФС

Статус задач: `Запланировано`, `В работе`, `Согласовано`, `Завершено`.

## Текущая итерация

### Разработка глоссария

Статус: В работе

Цель: последовательно согласовать официальные определения ключевых терминов СФС и зафиксировать их в `docs/0000-Принципы-СФС.md`.

Порядок терминов:

1. Семейная финансовая система (СФС) — завершено.
2. Семейный капитал — следующий термин для согласования.
3. Финансовая цель.
4. Фонд.
5. Портфель.
6. Актив.
7. Инвестиционная политика.
8. Проект.
9. Операционный бюджет.
10. Операционный резерв.

Для каждого термина обязательно указывается определение. Остальные разделы добавляются только при необходимости:

- назначение;
- основные свойства;
- связь с другими терминами;
- примеры;
- примечания.

## Завершённые уточнения документа принципов

- Удалена цель «Обеспечение преемственности системы».
- Удалено преждевременное указание операционного резерва как механизма финансовой устойчивости.
- Принцип уточнён до формулировки «Финансовая цель определяет способ её достижения».
- Удалён принцип «Финансовые цели определяют фонды».
- Связи СФС с будущими сущностями оставлены на уровень архитектуры.
- Перечень ещё не определённых терминов удалён из нормативного документа и ведётся в настоящем файле.
- Раздел «Критерии успешности» удалён из документа принципов.
