# Очередь работ по СФС

Статус задач: `Запланировано`, `В работе`, `Согласовано`, `Завершено`.

## Текущая итерация

### Разработка глоссария

Статус: В работе

Цель: последовательно согласовать официальные определения ключевых терминов СФС и зафиксировать их в `docs/0000-Принципы-СФС.md`.

Порядок терминов:

1. Семейная финансовая система (СФС) — завершено.
2. Активы — завершено.
3. Обязательства — завершено.
4. Чистый семейный капитал — завершено.
5. Финансовая цель — следующий термин для согласования.
6. Фонд.
7. Портфель.
8. Инвестиционная политика.
9. Проект.
10. Операционный бюджет.
11. Операционный резерв.

Для каждого термина обязательно указывается определение. Остальные разделы добавляются только при необходимости:

- назначение;
- основные свойства;
- связь с другими терминами;
- примеры;
- примечания.
