# Excel-модель

Статус: Draft

Excel-модель является реализацией архитектуры СФС и не является источником архитектурных решений.

Документ будет содержать:

- структуру листов;
- справочники;
- входные параметры;
- сценарии;
- показатели фондов;
- визуализации;
- контрольные панели;
- контроль соответствия модели документации СФС.

Изменения в Excel-модели выполняются только после согласования архитектуры и обновления документации.
