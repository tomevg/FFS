# Проекты СФС

Статус: Draft

## 1. Назначение документа

Документ устанавливает нормативную модель проекта СФС. Детальные правила жизненного цикла, закрытия и распределения остаточных активов будут определены в следующих итерациях.

## 2. Определение проекта

Официальное определение проекта приведено в глоссарии `0000-Принципы-СФС.md`.

## 3. Место проекта в архитектуре СФС

Проект является механизмом накопления активов для достижения конечной финансовой цели. Операции, не предусматривающие накопления и управляемого портфеля, проектами СФС не являются.

Активы принадлежат семье; проект задаёт их назначение и правила управления.

## 4. Финансовая цель проекта

Каждый проект обслуживает ровно одну явно моделируемую конечную финансовую цель.

Компоненты бюджета проекта не становятся отдельными финансовыми целями, если для них не требуется самостоятельное управление.

## 5. Портфель проекта

Каждый проект имеет ровно один отдельный портфель. Портфель проекта не может одновременно относиться к другому проекту, фонду или операционному резерву.

## 6. Инвестиционная политика проекта

Каждый проект использует ровно одну инвестиционную политику. Политика может использоваться другими механизмами при совместимости требований, но их портфели остаются раздельными.

## 7. Финансирование проекта

Источники, график и правила финансирования проекта устанавливаются в его паспорте. Перечисление средств из операционного бюджета в проект является внутренним распределением.

## 8. Бюджет проекта

Бюджет описывает состав и финансовую оценку результата проекта. Его компоненты не считаются самостоятельными целями без необходимости независимого управления.

## 9. Контроль достижения цели

Проект должен иметь измеримый критерий достижения конечной финансовой цели. Детальный состав показателей требует проектирования.

## 10. Жизненный цикл проекта

Состав стадий, статусов и переходов требует проектирования.

## 11. Завершение и закрытие

Основания и порядок завершения или досрочного закрытия проекта требуют проектирования.

## 12. Распределение остаточных активов

Правила распределения остаточных активов после закрытия проекта требуют проектирования.

## 13. Связи с другими документами

Проект связан с архитектурой СФС, инвестиционной политикой, финансовыми потоками и операционным бюджетом.
