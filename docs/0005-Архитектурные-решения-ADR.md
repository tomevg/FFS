# Архитектурные решения (ADR)

Статус: Draft

## Формат записи

```text
ADR-XXXX
Статус:
Контекст:
Решение:
Последствия:
```

## Начальные решения для последующей формализации

- Разделение фондов детей.
- Выделение операционного резерва.
- Отказ от фонда возможностей.
- СФС определяется как совокупность принципов, правил, архитектурных решений и процессов, а не как Excel-модель.
- Использование термина «портфель» вместо «инвестиционная корзина».
