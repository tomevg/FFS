# Семейная финансовая система (СФС)

Проект по разработке архитектуры персональной системы управления семейным капиталом.

## Назначение проекта

Создать долгосрочную семейную финансовую систему, основанную на принципах, правилах, архитектурных решениях и документации, а не на конкретных инвестиционных инструментах или программных средствах.

СФС — не Excel-таблица. Excel, приложения, базы данных, ИИ-ассистенты и другие программные средства являются реализацией системы и могут изменяться без изменения самой СФС.

## Правила разработки

Работа ведётся в последовательности:

> Проблема → Требования → Архитектура → Правила → Реализация

Документация является единственным источником истины.

Любое изменение проходит жизненный цикл:

1. Обсуждение проблемы или требования.
2. Согласование архитектурного решения.
3. Обновление соответствующей документации.
4. Обновление `CHANGELOG.md`.
5. При необходимости — создание или изменение ADR.
6. Только после этого — изменение Excel-модели или другой реализации.

Принятые архитектурные решения не изменяются без отдельного обсуждения.
