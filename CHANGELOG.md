# CHANGELOG

## v0.10

- Термин «Семейный капитал» исключён из очереди разработки глоссария как неоднозначный.
- В глоссарий добавлен термин «Активы» с категориальным подходом к признанию существенных объектов.
- В глоссарий добавлен термин «Обязательства», включая критерии признания экономически неизбежных будущих платежей.
- В глоссарий добавлен термин «Чистый семейный капитал» как разница между совокупной стоимостью активов и величиной обязательств на определённую дату.
- Зафиксировано, что чистый семейный капитал рассчитывается по всем признанным активам и обязательствам семьи.
- В `.chatgpt/TODO.md` следующим термином для согласования назначена «Финансовая цель».
- В бэклог добавлен рефакторинг документации после завершения глоссария: замена термина «семейный капитал» на точные термины по контексту.
- Из `.chatgpt/TODO.md` удалён раздел с историей завершённых уточнений; история изменений ведётся только в `CHANGELOG.md`.

## v0.9

## v0.8

- Из целей существования СФС удалена формулировка «Обеспечение преемственности системы».
- Из раздела целей удалено преждевременное указание операционного резерва как механизма финансовой устойчивости.
- Принцип «Сначала определяется цель, затем способ её достижения» уточнён до формулировки «Финансовая цель определяет способ её достижения».
- Удалён принцип «Финансовые цели определяют фонды» как преждевременно фиксирующий архитектуру.
- Связь СФС с другими сущностями заменена на нейтральное указание, что эти связи определяются архитектурой системы.
- Правило глоссария изменено: обязательным является только определение, остальные разделы добавляются при необходимости и не заполняются искусственно.
- Перечень ещё не определённых терминов удалён из нормативного документа и перенесён в `.chatgpt/TODO.md`.
- Раздел «Критерии успешности» удалён из `docs/0000-Принципы-СФС.md`.
- В `.chatgpt/WORKFLOW.md` добавлено правило соблюдения уровня абстракции каждого документа.
- В `.chatgpt/WORKFLOW.md` добавлено правило: нормативные документы не содержат планы, черновики, незавершённые сущности и пометки `TODO`; такие сведения ведутся только в `.chatgpt/TODO.md`.
